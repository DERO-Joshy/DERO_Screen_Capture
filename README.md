# DERO Screen Capture

![DERO Screen Capture Tool](https://github.com/DERO-Joshy/DERO_Screen_Capture/blob/master/Screenshot.png)


## Features
* Screen Grab
* Draw on image after capture
* Brush colour choice
* Brush Size Choice
